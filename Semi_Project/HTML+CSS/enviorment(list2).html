[환경일보] "바닷속 거북이를 살려라"




우리가 편하게 사용했던 빨대가 바다 속 친구들에겐 가장 무서운흉기가 되어 버렸던것입니다.


환경문제에 관심을가져야할 필요성이있다 우리가 일회용품과 관련해서 접근할 때 일회용품이 재질이 무엇인가 하는 것은 부차적인 문제입니다. 굳이 여러번 사용하지 않거나 다회용기를 쓸 수가 있음에도 불구하고 편리하다는 이유만으로 한 번 쓰고 버리는 일회용 문화 자체가 자원을 낭비하고 폐기물 발생량을 증가시키는 문제가 있습니다. 그렇기에 일회용 사용을 줄이자, 규제하자 라고 하는 것인데 종이니까 괜찮다라고 하는 것은 일회용품 사용의 본질을 왜곡시키는 문제가 있다라고 생각합니다
